# Launcharr Release Announcement

## Headline

Launcharr `v0.2.57` is now available

## Short Summary

Launcharr `v0.2.57` adds Houndarr as a built-in app with widget stats, fixes the `slskd` widget blank issue when a stale API key is stored, and resolves a settings page flash on navigation.

## What's New

- **Houndarr integration**: Houndarr ships as a first-class opt-in default app in the Arr Suite category. Widget stats show active instances, searches today, and total items found, with session-cookie auth against `/api/status`.
- **`slskd` widget fix**: the widget no longer returns blank when an invalid or expired API key is configured — it now falls back to username/password token auth automatically. The missing optional API key field has also been added to both settings pages.
- **Settings flash fix**: navigating to settings no longer briefly shows the General tab before jumping to the correct stored tab.
