# Launcharr v0.2.4

## [v0.2.4] - 2026-02-15

### Added

- Added custom theme toggles for sidebar invert, square corners, animated background, and free carousel scroll.
- Added ARR Calendar as a configurable overview section, including combined ARR calendar support.
- Expanded default app and overview configuration coverage, including Bazarr defaults.

### Changed

- Improved free-scrolling carousel behavior across ARR, Plex, Pulsarr, and Tautulli overview modules.
- Improved mobile UX for sidebar/nav behavior, launch-page frame sizing/spacing, and calendar continuity.
- Updated service worker cache strategy to reduce stale assets after upgrades.

### Fixed

- Hardened ARR proxy and fallback request paths with timeout handling.
- Added structured client/server error logging for ARR proxy/fallback failures.

### Migration Notes

- No schema migration required.
- Review custom theme toggles in `Settings` after upgrade if you want to enable new UI options.
- If using ARR Calendar modules (single or combined), verify the selected sources and overview placement match your layout.

### Verification Checklist

- [ ] Custom theme toggles save/apply correctly (sidebar invert, square corners, animated background, free carousel scroll).
- [ ] ARR Calendar renders in overview pages and combined calendar mode loads expected sources.
- [ ] Bazarr default app/overview entries appear correctly in manager/config flows.
- [ ] Free-scrolling carousel behavior remains smooth across supported overview modules.
- [ ] Mobile sidebar/nav and launch-page frame layout behave correctly across common screen sizes.
- [ ] ARR proxy timeout/fallback errors are handled gracefully and produce structured logs.
- [ ] Service worker refresh behavior does not leave stale assets after upgrade.
