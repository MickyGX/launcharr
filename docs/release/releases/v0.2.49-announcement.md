# Launcharr Release Announcement

## Headline

Launcharr `v0.2.49` is now available

## Short Summary

Launcharr `v0.2.49` polishes day-theme dashboard readability and preserves the previous Launcharr icon artwork as a bundled legacy asset.

## What's New

- Updated day-theme dashboard system chips with higher-contrast text, icon, and progress-bar styling.
- Updated the day-theme dashboard search bar to improve text and hover-state readability.
- Updated the day-theme dashboard links widget so titles and link labels stay readable on light backgrounds.
- Added a bundled `launcharr-icon-old.png` asset for legacy branding and release-material reuse.

## Why It Matters

- Day-theme deployments are easier to scan without washed-out dashboard controls.
- Search and quick-link widgets stay readable in lighter dashboard configurations.
- The previous Launcharr icon remains available when older branding is needed for docs or announcements.

## Upgrade Notes

- Docker image: `mickygx/launcharr:v0.2.49`
- Supplementary tag: `mickygx/launcharr:development`
- No migration steps required.
- Rebuild/restart Launcharr after upgrade so the updated assets and dashboard CSS are served.
