# Launcharr Release Announcement

## Headline

Launcharr `v0.2.56` is now available

## Short Summary

Launcharr `v0.2.56` adds built-in `slskd` support, introduces configurable dashboard auto-refresh, and fixes CSRF validation failures on valid login/setup and settings actions.

## What's New

- **Built-in `slskd` support**: `slskd` now ships as a first-class downloader integration with default app metadata, icon support, widget stats, settings auth options, and dashboard/app overview queue handling.
- **Dashboard auto-refresh**: each dashboard can now opt into a configurable refresh interval so supported overview cards and queue modules update in place without a full page reload.
- **CSRF coverage fixes**: Launcharr now injects CSRF protection consistently for local login/setup forms, Plex PIN registration, and JavaScript-generated settings forms, fixing false `CSRF validation failed.` screens during legitimate actions like dashboard item removal.
