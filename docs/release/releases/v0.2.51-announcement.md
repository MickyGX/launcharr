# Launcharr Release Announcement

## Headline

Launcharr `v0.2.51` is now available

## Short Summary

Launcharr `v0.2.51` focuses on Arr widget efficiency, release-build hygiene, and general code cleanup for a cleaner, lighter release.

## What's New

- Reduced internal widget polling cost for Radarr, Sonarr, Lidarr, and Readarr by using lightweight status checks during background monitoring.
- Added cached Arr summary metrics so widgets can retain useful counts without repeatedly walking full libraries.
- Cleaned up release packaging rules so local-only development files stay out of public images and source history.

## Why It Matters

- Arr-backed dashboards put less routine pressure on Radarr, Sonarr, Lidarr, and Readarr.
- Public release images are cleaner and closer to the published source layout.
- Operators get a quieter, more predictable Launcharr background monitor path on larger libraries.

## Upgrade Notes

- Docker image: `mickygx/launcharr:v0.2.51`
- Supplementary tags: `mickygx/launcharr:development`, `mickygx/launcharr:latest`
- No migration steps required.
- Hard refresh the browser after upgrade so the updated widget runtime is loaded.
