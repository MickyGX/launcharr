# Launcharr Release Announcement

## Headline

Launcharr `v0.2.58` is now available

## Short Summary

Launcharr `v0.2.58` adds Jellyfin synced login and Jellyfin user role management, introduces per-provider login visibility toggles for Plex and Jellyfin, and tightens login/sidebar polish across the UI.

## What's New

- **Jellyfin synced login**: users can now sign into Launcharr directly from the login page with Jellyfin credentials, and Launcharr resolves their synced role from the existing admin/co-admin mappings.
- **Jellyfin role management**: the new `Jellyfin Users` settings card fetches Jellyfin accounts, shows recent activity plus Launcharr login history, and lets admins assign Launcharr roles without editing config files.
- **Login visibility controls and UI fixes**: Plex and Jellyfin can now be hidden from the login screen independently, new unconfigured providers default to off, and the login page behaves better on shorter screens.
