# Changelog

## [v0.2.42] - 2026-02-27

### Added

- Added a major System Info widget expansion (`sys-resources`) with configurable CPU, RAM, multi-disk, search, and weather elements in the widget settings canvas.
- Added a `Show Total Space (RAM + Disks)` widget option so System Info cards can switch between free-space and total-space displays.
- Added widget status notifications via Apprise with dedicated settings controls:
  - `Enable widget status alerts`
  - `Widget status delay (seconds)`
- Added a background widget-state monitor that polls widget apps and emits online/offline notifications only after the configured delay threshold is met.
- Added a widget monitor debug surface:
  - Admin API endpoint: `GET /api/widget-status-monitor`
  - Live monitor summary line in `Settings -> Log` showing tracked apps and pending transition timers.
- Added dedicated system widget icon assets (`cpu`, `memory`, `disk`, weather-condition icons) and a dedicated dashboard widget runtime script (`public/widgets-stat-cards.js`).

### Changed

- Updated widget row controls in `Settings -> Custom -> Widgets`:
  - Replaced row add CTA flow with compact up/down row insertion strips.
  - Reduced left-side row action footprint for denser editing.
- Updated widget add-menu layering/placement so floating menus and widget popups remain visible above surrounding settings/topbar UI.
- Updated System Info widget visual treatment to align with the dashboard theme language (single themed outer container with glow treatment, dark panel styling, and reduced inner framing noise).
- Updated System Info responsive layout behavior, including mobile wrapping and clearer search/weather placement consistency.
- Updated widget behavior to allow multiple System Info widgets in the same widget bar configuration (removed single-instance restriction).

### Fixed

- Fixed widget status transition handling so transient `unknown` probe results do not prematurely reset pending online/offline notification timers.
- Fixed System Info disk presentation consistency when multiple disk paths are configured, including OS/media path rendering behavior.
- Fixed widget preview/runtime parity issues between settings canvas previews and dashboard widget output.

### Migration Notes

- No schema/data migration required.
- To use widget status notifications:
  - Enable Apprise notifications.
  - Enable widget status alerts.
  - Set your delay threshold (seconds).
- Rebuild/restart Launcharr and hard-refresh browser assets once after upgrade to ensure latest widget scripts/styles are loaded.
