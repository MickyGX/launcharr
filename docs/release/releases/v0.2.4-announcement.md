# Launcharr v0.2.4 Release Announcement

## Headline

Launcharr `v0.2.4` is now available.

## Short Summary

This release adds custom theme toggles, introduces ARR Calendar overview modules (including combined mode), and improves mobile/overview behavior with proxy/cache reliability fixes.

## What's New

- Added custom UI theme toggles for sidebar invert, square corners, animated background, and free carousel scrolling.
- Added ARR Calendar as an overview module with combined ARR calendar support.
- Expanded default app/overview configuration coverage, including Bazarr defaults.
- Improved carousel scrolling behavior and mobile layout continuity across overview and launch surfaces.
- Hardened ARR proxy/fallback handling with timeout protection and structured error logging.
- Updated service worker cache behavior to reduce stale frontend assets after upgrades.

## Why It Matters

- Admins get more control over visual behavior without editing styles manually.
- ARR calendar activity can be surfaced more flexibly in overview dashboards.
- Mobile navigation and launch interactions are more consistent across devices.
- Upgrade reliability improves with better caching behavior and clearer proxy failure diagnostics.

## Upgrade Notes
