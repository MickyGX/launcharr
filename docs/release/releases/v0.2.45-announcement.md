# Launcharr Release Announcement

## Headline

Launcharr `v0.2.45` is now available

## Short Summary

Launcharr `v0.2.45` delivers touch interaction fixes for carousel and page scrolling in mobile-width layouts on desktop touchscreen devices, plus updated mobile launch-flow documentation.

## What's New

- Fixed carousel swipe handling so pointer capture is limited to mouse drags across overview modules.
- Fixed vertical page scrolling in mobile-width Launcharr layouts on desktop touchscreen hardware.
- Updated widget-row gesture handling (`touch-action: pan-x pan-y`) to preserve both horizontal row movement and vertical page panning.
- Added mobile launch-flow GIF docs and wiki references.

## Why It Matters

- Desktop touch users can now scroll pages vertically in mobile-width views without losing carousel swipe controls.
- Immich and other overview carousels behave more consistently across touch-capable browsers.
- Admin/user docs now include a dedicated mobile launch-flow visual.

## Upgrade Notes

- Docker image: `mickygx/launcharr:v0.2.45`
- Supplementary tags: `mickygx/launcharr:development`, `mickygx/launcharr:latest`
