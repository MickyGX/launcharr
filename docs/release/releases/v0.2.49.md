# Changelog

## [v0.2.49] - 2026-03-17

### Added

- Added a legacy `launcharr-icon-old.png` asset so the previous Launcharr branding remains available for documentation and release collateral.

### Changed

- Updated day-theme dashboard system chips to use higher-contrast backgrounds, labels, values, bars, and icons.
- Updated the day-theme dashboard search bar with brighter surfaces and darker input/hover states for better readability.
- Updated day-theme dashboard links widgets so titles, names, and initials stay readable against light card backgrounds.

### Fixed

- Fixed washed-out day-theme dashboard status chips that were difficult to read on light backgrounds.
- Fixed low-contrast text in day-theme dashboard search and links widgets.

### Migration Notes

- No schema or config migration required.
- Rebuild/restart Launcharr after upgrade so the updated dashboard styling and legacy icon asset are included in the served bundle.

### Verification Checklist

- [ ] Day theme dashboard system chips remain readable for values, labels, icons, and progress bars.
- [ ] Day theme dashboard search input and button states are readable in idle and hover states.
- [ ] Day theme dashboard links widget titles, names, and initials remain legible on light cards.
- [ ] `public/icons/launcharr-icon-old.png` is present in the built image and available for documentation/release use.
