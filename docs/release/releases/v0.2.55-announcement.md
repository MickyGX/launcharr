# Launcharr Release Announcement

## Headline

Launcharr `v0.2.55` is now available

## Short Summary

Launcharr `v0.2.55` fixes broken login/logout redirects for users hosting under a subdirectory reverse proxy (e.g. Nginx Proxy Manager), and adds status pills and Request buttons to Pulsarr/Seerr most-watchlisted cards.

## What's New

- **Base-path redirect fix for NPM and subdirectory proxies** — all auth redirects (`/login`, `/dashboard`, `/setup`, logout) now correctly include the configured base path. Users behind Nginx Proxy Manager (or any proxy routing by path prefix) were getting 404s or redirect loops on login and logout since v0.2.54. This release resolves it.
- **Seerr most-watchlisted status pills** — most-watchlisted cards now show the same coloured status pills (Available, Requested, Processing, Partial) and Request button as search results, so you can see at a glance what's already in your library and request what isn't.
- **Settings page auth guard fix** — visiting `/settings` while logged out under a base path now correctly redirects to login instead of returning a 403.
